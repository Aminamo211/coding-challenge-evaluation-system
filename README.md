# Coding Challenge Evaluation System

## Overview
This is a Java-based system designed to create and evaluate coding challenges.  
The system allows users to define problems, add test cases, and evaluate outputs against expected results.

## Features
- Create coding challenges with title, description, and difficulty
- Add multiple test cases for each challenge
- Evaluate user output against expected output
- Generate pass/fail results with feedback
- Object-oriented design using multiple classes

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- IntelliJ IDEA
- Git & GitHub

## How It Works
1. Create a coding challenge  
2. Add test cases (input + expected output)  
3. Submit a solution output  
4. System evaluates and returns pass/fail result  

## Example
Input: hello  
Expected Output: olleh  
User Output: olleh  
Result: Passed  

## Project Structure
- CodingChallenge.java → stores challenge details  
- TestCase.java → stores input/output pairs  
- Submission.java → stores user results  
- Evaluator.java → checks correctness and generates feedback  
- Main.java → runs the program  

## Future Improvements
- Support multiple test case evaluation (score system)  
- Save data using files or database  
- Add graphical user interface (JavaFX)
